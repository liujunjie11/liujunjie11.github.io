Hexo博客图片外链从七牛云移至腾讯云

发表于 2018-11-11 | 分类于 笔记 ， Hexo | |
字数统计: 812 | 阅读时长 ≈ 2

最近发现博客中的一些文章图片无法正常显示了，查看了一些文章之后，发现原来是测试域名失效了…我的图片存放了这么多…全都不能用了，妈蛋啊！

我研究了一下关于它的这个恢复的方法，需要实名不说了，还要我的网站备案，不可能的绝对不可能的，我要走了，再见了，七牛云！！

我要搬去腾讯云了。以下是我的记录。
