  • 一是pyQt5这个库，结合Qt creator来进行完成。

  • 另外就是通过网页形式来进行交互式的实现了。

在接触了前者之后发现学习的时间成本比较高一些，所以打算选择后者了…

安装

在终端使用命令行：pip3 install Django

默认python3进行。python2:pip install Django。两者均会下载Django库的最新版本，亦可指定下载那个版本。
